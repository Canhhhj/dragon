# Dragon Activiti - Cập nhật Super Sale giảm %

## Chức năng đã thêm
- **Quản lý giảm giá siêu khuyến mãi**: Có thể nhập % giảm cho từng sản phẩm siêu sale.
- **Hiển thị tự động**: Website sẽ hiển thị badge giảm giá và giá sau khi giảm.

## Cách chạy
```bash
npm install
npm run dev
```

Trang Admin: `http://localhost:8083/admin.html`
Trang Web: `http://localhost:8080` hoặc `http://localhost:8082`
